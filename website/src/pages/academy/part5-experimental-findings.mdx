## 1. Histogram Invisibility & Head Distribution

**Finding:** Head function cannot be read off static weight geometry. Retrieval and Induction heads are statically indistinguishable from Local heads, showing up only under a dynamic behavioral probe.

Through dynamic profiling of **1,568 total attention heads** across GPT-2, Qwen-0.5B, Qwen-1.5B, and Llama-3.2-1B, we extracted the following universal distribution:
- **Local Heads:** 51.0% (800 heads)
- **Sink Heads:** 41.1% (645 heads)
- **Induction Heads:** 6.9% (108 heads)
- **Retrieval Heads:** 1.0% (15 heads)

## 2. The V/Q Spatial Scaling Law (WEAKER THAN EXPECTED)

**Hypothesis (Law 1):** Deeper heads become heavily biased toward Value vectors (higher V/Q ratio) and causally exert massive output norms into the residual stream.

**Experiment:** We performed Partial Correlation controlling for layer depth.
**Result:** Raw Pearson correlation is strong across all four architectures (r = 0.48–0.71, pooled p = 1.92×10⁻¹²⁷), but a large fraction of that is confounded by depth. Controlling for it via partial correlation, the effect ranges from negligible in GPT-2 (r = 0.09, p = 0.08) to moderate in Llama (r = 0.45). 

**Conclusion: WEAKENED.** Real, but smaller and less universal than the headline number suggests.

## 3. The Early/Late Induction Split (PROVEN)

**Finding:** Induction heads are not monolithic. They cleanly separate into depth-separated **Early (Locate)** and **Late (Copy-Forward)** subpopulations.
**Result:** This split is statistically sharp across architectures (Mann-Whitney p = 4.17×10⁻³²).

## 4. Retrieval-Induction Co-Gating (FALSIFIED)

<Misconception>
  <Fragment slot="myth">Retrieval heads form a strict bottleneck for Induction heads. If you ablate Retrieval heads, the model loses its ability to copy facts.</Fragment>
  <Fragment slot="truth">The reasoning circuit is massively redundant, parallelized, or routes through heads we haven't identified yet.</Fragment>
</Misconception>

**Experiment:** We ran an apples-to-apples causal ablation on Qwen2.5-1.5B across 5 diverse Needle-In-A-Haystack prompts. We forcefully zeroed-out the `o_proj` weight matrices for **all 6 known Retrieval heads simultaneously**.

**Result:**
- **Logit Drop:** The probability of predicting the correct answer dropped by only **8.36% ± 9.03%**.
- **Conclusion: FALSIFIED.** This is a noisy drop that is not clearly distinguishable from zero. The original single-pair ablation result doesn't replicate at the full-circuit level. The retrieval-induction interaction is real but far more redundant/parallelized than a strict AND-gate.

## 5. Polysemantic Multiplexing (PROVEN BY SPARSITY, NOT INTERPRETABILITY)

**Experiment:** We trained a 4x overcomplete Micro-SAE. As a rigor check, we trained a **Null-SAE** on temporally shuffled vectors to destroy true covariance. Both were evaluated on a held-out passage of Python code.

**Result:** 
- **The Sparsity Gap (Proven):** The True SAE required only **4.5 active features** per token. The Null SAE required **51.0 active features**. 
- **The Qualitative Falsification:** Initially, features cleanly tracked rare abstract tokens like `def` or `_search`. However, the **Null SAE also perfectly tracked these exact same rare tokens**. Qualitative interpretability on rare tokens is a statistical illusion.

**Conclusion:** The only rigorous, surviving evidence of multiplexing is the massive L0 sparsity gap, not qualitative feature tracking. 

## 6. Validating Atlas Roles via Attention Routing

We built a native routing engine to intercept head outputs and force them into highly constrained attention kernels.

![Developmental Sankey](/attentionheadgenome/images/final_artifacts/developmental_sankey.png)
*Figure: The developmental trajectories of functional head species across network depth.*

- **Local Head Success:** We forced all Local heads into a strict **32-token sliding window**. HellaSwag accuracy dropped by only **1.0%**. The atlas mapping is accurate.
- **Sink Head Falsification:** We forced Sink heads (which already dump >50% mass on BOS) to attend *exclusively* to the BOS token. We hypothesized this cheap rest state would be safe. Instead, **accuracy catastrophically collapsed to 26% (random chance).** Sink heads perform critical global integration.

## 7. Unsupervised Emergent Discovery

![Second Axis Curve](/attentionheadgenome/images/final_artifacts/second_axis_curve.png)
*Figure: The UMAP/HDBSCAN clustering proving the existence of a continuous megacluster.*

**Result:** Unsupervised clustering revealed that 923 heads (58%) collapsed into a **Giant Megacluster**. The boundaries between Local, Sink, and Induction roles are highly continuous, forming a developmental manifold rather than discrete buckets.
